About me 私について

鈴木 宏樹

1993年生まれ。芝浦工業大学を卒業後、東京消防庁に入庁し、夢だった特別救助隊として10年間勤務しました。その後、web制作に興味を持ちWeb制作スクールに8ヶ月間通いました。クライアント様の要望を丁寧に汲み取り、確実に形にすることを大切にしています。 そのうえで、より良い成果を生み出すための提案や改善案も積極的にお伝えします。 「依頼通り」だけでなく、「期待以上」の仕上がりを目指し、二人三脚でプロジェクトを進めていきます。

Skill 出来ること

  • HTML

    Webページの基礎となる構造を正しく組み立てることを重視しています。 SEOやアクセシビリティを意識したセマンティックなマークアップを行い、誰が見ても理解しやすく、検索エンジンにも適切に評価されるコードを書くことを心がけています。

  • CSS

    FlexboxやGridを活用し、レスポンシブで美しいレイアウトを設計します。 配色や余白のバランス、タイポグラフィなど、細部までこだわったデザインを実装します。アニメーションやトランジションで動きのある表現を加え、サイト全体の印象を高めます。

  • JavaScript

    ES6以降のモダンな構文を用い、動きや機能を持たせたWebサイトを実装します。 DOM操作や非同期処理による動的コンテンツの生成、フォームバリデーション、UIアニメーションなど、ユーザー体験を向上させるための仕組みを構築します。

  • WordPress

    静的デザインをもとにオリジナルテーマを制作し、WordPressとして運用できる形に構築します。 カスタム投稿タイプやカスタムフィールド（ACF）を活用し、クライアントが管理画面から簡単に更新できるサイトを作ります。
